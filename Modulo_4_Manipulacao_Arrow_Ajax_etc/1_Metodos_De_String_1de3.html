[JS] - Certo desenvolvedor precisou escrever um trecho de código para formatar uma string de 11 caracteres num cpf tradicional (000.000.000-00). Observe abaixo: let str = "06484966825"; let arrStr = str.[A]; let strtamplate = `${arrStr[0]}${arrStr[1]}${arrStr[2]}. ${arrStr[3]}${arrStr[4]}${arrStr[5]}. ${arrStr[6]}${arrStr[7]}${arrStr[8]}- ${arrStr[9]}${arrStr[10]}`; console.log(strtamplate); Marque a opção do método [A] no código acima que seja correto para o retorno funcionar. Observação: O trecho de código acima é apenas para fins didáticos. Na vida real, um código que realize uma operação como esta deve estar dentro de uma função de escopo local, e dentro da mesma possuir verificações e validações da string (argumento) que será convertida.
